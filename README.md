# DSP_DSK6416
Audio Processing Studies and Digital Communications with Digital Signal Processor TMS320C6416T

## Target:
### [DSP TMS320C6416T](http://www.ti.com/lit/ds/symlink/tms320c6416.pdf)
Fixed-point DSP, 8000 MIPs

Frequency: 8 kHz, 16 kHz, 24 kHz, 32 kHz, 44.1 kHz, 48 kHz and 96 kHz

Amplitude: 6 Vpp

Word-lenght: 2 channels witch 16-bit each.

### DSP Starter kit [TI DSK 6416](http://www.ti.com/tool/TMDSDSK6416)
CODEC: AIC23, Implements anti-aliasing, DAC and ADC.
